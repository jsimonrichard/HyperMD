# ProseMark Demo Page

A _"What You See Is What You Mean"_ editor like **Typora** or **Obsidian**, built on top of [**CodeMirror 6**](https://codemirror.net/).

## Features

Basic formatting including _italics_, **bold text**, `code spans`, and ~~strike throughs~~.

  - Item 1.2
- Item 2

- [x] Task 1
- [ ] Task 2
